# Hospital Logo

## Logo Location

The hospital logo is located at: `src/assets/images/hospital-logo.png`

## Logo Specifications

- **Format:** PNG (with transparent background preferred)
- **Size:** At least 200x200 pixels (higher resolution is better)
- **Shape:** Square or circular images work best with the current component styling

## Supported Formats

- PNG (recommended)
- JPG/JPEG
- SVG

## Where the Logo Appears

The logo automatically appears on:
- Home Page
- Login Page
- Scheduled Appointments Page header
- Dashboard Page
- Navigation sidebar

## Note

If the logo file is not found or fails to load, a fallback SVG icon will be displayed automatically.
